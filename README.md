
 # Communication Scorer  
A simple AI-based tool that analyzes communication and assigns a score based on predefined rubric rules.  
Backend runs on Python (FastAPI/Flask), and frontend is built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- Score calculation based on rubric values  
- Easy-to-use web interface  
- Lightweight backend with simple logic  
- Clean project structure  
- No large dependencies or ML models required

---
